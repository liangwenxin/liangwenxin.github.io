1、Vim编辑器的模式

  • 正常模式 (Normal-mode)
  • 插入模式 (Insert-mode)
  • 命令模式 (Command-mode)
  • 可视模式 (Visual-mode)
2、.vimrc 配置vim编辑器的配置

syntax on //语法高亮
set number //显示行号

3、移动光标

在命令模式下：可以使用h、j、k、l键分别代表方向键←↓↑→

w（word）：键盘可以以单词为单位跳
b（back）：键可以往回以单词为单位跳

ctrl+f（forward）：可以往下翻页 等于Pg up
ctrl+b（backward）：可以往上翻页 等于Pg down

数字+gg：可以以数字为行数跳到第数字行，10gg跳到第10行
数字+j、k：可以以数字为行数跳转，10j往下跳10行，10k网上跳10行

/查找的内容
n（next）：可以查找下一个
shitf+n（next）：查找上一个

cc：剪贴删除会自动跳入编辑模式
p：粘贴插入复制的行
yy（yank）：复制当前行，2yy复制两行
c数字c：删除数字的行数，c2c删除两行
dd：删除当前行，2dd删除两行

u（undo）：撤回操作

shift+g：跳到最后一行

3、视图模式

c：剪切
p：粘贴

ctrl+n：自动补全

vim plug自动补全插件
