循环队列的顺序存储结构

队空的条件：Q.front == Q.rear
队满的条件：(Q.rear+1)%MAXQSIZE == Q.front
求队列长度：(Q.reat-Q.front+MAXQSIZE)%MAXQSIZE
入队：rear++（尾指针）
出队：front++（头指针）
